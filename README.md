## [SEU抢讲座](SEUjiangzuo.js)
脚本相对于网页图形，最大的优点是不用刷新页面（好几次都是因为刷新导致卡死）
在ACupofAir的基础上修改。增加了以下特性：
- 时间缺省时自动读取表格得到最早预约时间
- 便于连续抢
    - 去掉了alert，增加一个div显示信息；
    - 抢完自动刷新验证码
    - 验证码输完，按回车就可以抢
    - 增加一个立即抢的按钮
- 倒计时用了更精准的方式

注意，东南大学疑似使用国家授时中心时间；而电脑默认的windows授时中心一般会快1~2秒（问就是准点请求但是返回值是“还没开始”）。所以建议使用`time1.aliyun.com`。

使用方法：
- 先填好验证码（验证码可以提前获取！），一个验证码只能照顾下一个请求。
- 选择要预约的讲座，直接点击“定时抢课”，等就好了

## [智慧树自动刷课脚本](zhihuishuShuake.js)
针对《创新思维与创业实验》这个慕课开发。

如题，16倍速、低清晰度播放智慧树视频，并自动跳转下一个视频。

仅仅适配了深色背景的、左侧有竖着的菜单的（有不同的课程详情）

和雨课堂相比，智慧树的视频后台播放也算数。虽然智慧树有防脚本，但是只要设置`// @run-at document-start`就可以绕过了。雨课堂有现成的自动刷脚本，但是最高两倍速（手动设置16倍速会卡死）
