# 《创新思维与创业实验》题库收集系统
分为三个部分：
1. 前端本地数据库 油猴插件 zhihuishuTask.js
2. 后端云端数据库 backend.js
3. 后端数据导出 exportCSV.js

## 前端
自己整理太费时间了，所以拦截网络请求，直接抓取题目，并实现了做对的题自动答、做错的记录历史答案的功能，极大增加了答题速度。

智慧树的前端有些鸡贼的，但是还是我魔高一丈啊！哈哈哈！

用Web API 自带的 indexDB 存储题目，并向后端发送网络请求进行同步。

## 后端
本意是实现互利共赢，结果只有室友帮我贡献进度。用简单的sql+原生https功能实现，AI写了大部分代码。

注意由于智慧树是HTTPS，因此后端必须有HTTPS证书。巧了，刚好有一个域名，刚好SEU校园网的80端口是开的，美滋滋部署。

## 导出
后端遍历 `.db` 得到CSV，在VSCode里面用GB2312保存，然后用Excel打开。
